# Кабельный якорь

<GameScene zoom="6" background="transparent">
  <ImportStructure src="../assets/assemblies/cable_anchor.snbt" />
  <IsometricCamera yaw="195" pitch="30" />
</GameScene>

Маленькие декоративные шипы для предотвращения соединений кабелей, создания «лестниц» с кабелями или создания эффекта соединения кабеля
к стенам вокруг. Также используются для создания <ItemLink id="facade" />.

Кабельные якоря предотвращают образование соединений на той стороне, на которой они установлены.

По ним можно подниматься, как по лестнице.
