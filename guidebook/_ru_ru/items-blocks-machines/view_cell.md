# Ячейка просмотра

<ItemImage id="view_cell" scale="2" />

Ячейки просмотра используются для фильтрации отображения [терминалов](terminals.md). Вы разделяете их на разделы в <ItemLink id="cell_workbench" />.

Например, вы хотите, чтобы на терминале отображался только набор каменных строительных предметов. Разделите ячейку просмотра на эти
предметы и поместите её в терминал, и будут отображаться только эти предметы.

Ячейки просмотра являются аддитивными: если у вас есть ячейка просмотра для дубовых досок и ячейка просмотра для булыжника, то при добавлении обеих будут отображаться
и доски, и булыжник.
