# Беспроводные терминалы

<Row>
  <ItemImage id="wireless_terminal" scale="4" />

  <ItemImage id="wireless_crafting_terminal" scale="4" />
</Row>

Беспроводные терминалы — это портативные версии обычных проводных [терминалов](terminals.md). Они имеют точно такой же интерфейс, как и их
проводные аналоги, за исключением того, что вместо слотов для <ItemLink id="view_cell" />, у них есть слоты для [карт улучшения](upgrade_cards.md)

Чтобы подключить их к сети, поместите терминал в правый верхний слот <ItemLink id="wireless_access_point" />
подключенной к этой сети. (Слот с изображением беспроводного терминала и стрелкой под ним).

Для работы они должны находиться в зоне действия <ItemLink id="wireless_access_point" />.

Их можно зарядить с помощью <ItemLink id="charger" />а.

# Беспроводной терминал

<ItemImage id="wireless_terminal" scale="4" />

Ваш базовый терминал, теперь портативный! Просматривайте и получайте доступ к содержимому вашего [сетевого хранилища](../ae2-mechanics/import-export-storage.md)
и запрашивайте предметы из вашей системы [автоматического создания](../ae2-mechanics/autocrafting.md) из любой точки в пределах досягаемости
<ItemLink id="wireless_access_point" />.

# Беспроводной терминал создания

<ItemImage id="wireless_crafting_terminal" scale="4" />

Беспроводной терминал создания похож на обычный беспроводной терминал, с теми же настройками и разделами, но с добавленной сеткой для создания, которая будет автоматически
пополняться из [сетевого хранилища](../ae2-mechanics/import-export-storage.md). Будьте осторожны, нажимая на вывод, удерживая [Shift]!

## Пользовательский интерфейс

См. [терминалы](terminals.md).

## Улучшения

Беспроводной терминал создания поддерживает следующие [улучшения](upgrade_cards.md):

*   <ItemLink id="energy_card" /> для увеличения ёмкости аккумулятора.
